# Portfolio — Sudhanshu Kumar Suman

## Files in this folder

```
portfolio/
├── index.html   ← Page structure & content (edit this to update info)
├── style.css    ← All visual styling (edit this to change colors/fonts)
├── script.js    ← All interactive behavior (theme toggle, scroll, nav)
└── README.md    ← This file
```

---

## How to open locally
Just double-click `index.html` — it opens in any browser. No server needed.

---

## Common updates

| What to change | Where to look in index.html |
|---|---|
| Your name | Search: `hero-name` |
| Your role | Search: `hero-role` |
| Description | Search: `hero-desc` |
| CGPA | Search: `7.67` |
| Add a skill | Find the right `.skills-block`, copy a `.chip` line |
| Add a project | Copy a `.project-card` block inside `.projects-grid` |
| Add a certificate | Copy a `.cert-card` block inside `.certs-grid` |
| Add education entry | Copy a `.tl-item` block inside `.timeline` |
| Update contact links | Search: `contact-links` |

## How to change colors
Open `style.css` → find `THEME VARIABLES` at the top → change hex values.

## How to host (free)
1. Go to https://github.com → create a repo named `yourusername.github.io`
2. Upload all 3 files (index.html, style.css, script.js)
3. Your site is live at `https://yourusername.github.io`
